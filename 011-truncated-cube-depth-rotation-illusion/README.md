# 截角立方体深度旋转错觉

一枚由冷白微光点构成的透明截角立方体。普通立方体的八个顶点沿相邻三条棱各切去边长的 14%，形成八个小三角切面；这些切面使用略高的点密度成为运动参照，但不使用发光边线、实体表面或阴影。

物体在正交镜头下绕竖直轴以 12 秒一圈匀速旋转。约 9600 个细碎微光点让正面和背面的表面同时保持可读，削弱遮挡、明暗和透视线索，使观看者可能重新解释前后关系并感知到旋转方向反转。

最终成片为 24 秒、30 fps、720 × 1280，共包含两次完整旋转，并以循环后的下一帧与首帧重合的方式保持无缝首尾。

## 视觉设计

- 边长 `2.0`，切角深度 `0.28`，即 14%。
- 六个大八边形面使用随机均匀点阵，八个小三角切面保持相近的面积点密度。
- 所有表面共用同一种冷白发光材质，不用分面颜色。
- 所有棱两侧使用约 `0.035` 宽、降低约 22% 点密度的窄暗槽，以局部对比衬托轮廓而不是简单加粗。
- 十二条主要长棱使用带约 18% 随机断点的冷蓝紫双层星尘光丝：约 470 个核心点叠加约 95 个低透明柔光粒子，并保持约 ±6% 的整体呼吸。
- 二十四条截角短棱使用两端弱、中间强的珠状星点，每条短棱中央设一个微型亮点，呼吸幅度仅约 ±3%。
- 所有轮廓效果均不使用沿棱流动的方向性高光。
- 正交相机高于水平面 10°，只显露少量顶部结构。
- 正交范围为 `5.35`，完整容纳旋转到 45° 时的最大对角线投影。
- 无阴影、无环境遮蔽、无地面、无倒影、无景深。
