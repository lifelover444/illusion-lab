# 互锁双星环深度旋转错觉

两个真正互锁的扁椭圆点云环组成一件刚性星尘雕塑。两个环使用相同的冷白点色、点密度与亮度，在正交镜头下共同绕竖直轴旋转。稀疏点阵会弱化交叉处的实体遮挡，使观众可能同时重新解释旋转方向以及两个环的前后关系。

## 视觉设计

- 使用标准 Hopf 链式拓扑：两个等大圆环位于相互垂直的平面，并从彼此孔中穿过。
- 环体为约 2:1 的扁椭圆截面，不使用粗圆管。
- 每个环约 2304 个均匀扰动微光点，总数约 4608。
- 两个环使用完全相同的冷白发光材质。
- 无阴影、无环境遮蔽、无地面、无倒影、无分环颜色。
- 两环作为一个刚体匀速旋转，不添加独立滚动、漂浮或缩放。

## 命令

```powershell
npm run scene:010
npm run render:010:stills
npm run render:010:preview
npm run render:010:final
```

默认 Blender 路径为 `D:\software\Blender 4.5 LTS\blender.exe`，也可以通过 `BLENDER_EXE` 环境变量覆盖。
